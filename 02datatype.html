자바스크립트 자료형

변수(Variable)와 상수(constant)

변수는 데이터를 담는 그릇 - var, let으로 선언

상수는 변하지않는 값을 담는 그릇 - const로 선언

변수/상수 선언시 자료형을 기술할 필요 x

변수 선언시 var이나 let(ES6에서 추가)을 사용

실무에서는 오류를 최소화하기 위해 let사용을 추천!

자바스크립트에서는 각 문장의 끝은 ;으로 마무리해야 함

자료형

자바스크립트 자료형에는 Number, String, Boolean, Null, Undefined, BigInt, 객체형 등이 있음

숫자형

Number형으로 정수, 실수 등을 다룰 수 있음

BigInt형으로 기존 정수보다 더 많은 범위의 수를 다룰 수 있음

한편, Number형에는 일반적인 숫자외에 추가적으로 NaN(Not a Number), Infinity/-Infinity등의 특수숫자값을 사용할 수 있음

따라서 뭔 말도 안되는 이상한 연산에도 오류는 발생하지 않음

문자형

단일문자나 단일문자가 여러개 모인 문자열을 다룰때 사용

문자열을 정의할때는 '', ""을 이용함

템플릿 리터럴을 정의할때는 ``을 이용함

Boolean형

논리형의 일종으로 true, false 등의 논리값만 존재

주로 긍정(true), 부정(false)을 나타내는 값에 사용

Null

null 값만 다루는 자료형

null은 존재하지 않는 값, 비어있는 값, 알 수 없는 값을 나타내는데 사용

undefined

undefined 값만 다루는 자료형

undefined는 변수는 선언했지만, 값이 할당되지 않은 상태를 나타내는데 사용

객체형

String, Number, Boolean형은 단 하나의 값만 표현가능 - 원시(primitive) 자료형

하지만, 복잡한 프로그램의 경우 이것으로는 다소 부족

데이터의 집합이나 개체를 표현하기 위해 객체 자료형이 도입

즉, 변수 하나에 여러 종류의 데이터들을 담을 수 있음
